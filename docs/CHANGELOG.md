# Changelog

## [0.3.0] — 2026-02-13

### Добавлено

- Комплексная мобильная оптимизация (Responsive Design).
- Внедрение `dvh` для стабильной высоты секций на мобильных устройствах.
- Компактный хедер с интегрированным переключателем языков.
- Адаптивные сетки для всех компонентов (`Hero`, `Method`, `Competencies`, `LocalSolutions`).
- Полная локализация и перенос всех текстов в `src/i18n/ui.ts`.

---

## [0.2.0] — 2026-02-13

### Добавлено

- Полноценная поддержка мультиязычности (RU, EN, VI).
- Автоматическое определение языка пользователя на основе заголовков браузера.
- Компонент `LanguagePicker` для ручного переключения языков.
- Локализация всех блоков главной страницы и футера.
- Выделение секций «О себе» (`About`) и «Метод» (`Method`) в отдельные Astro-компоненты.

---

## [0.1.0] — 2026-02-13

- Система управления проектом: `BACKLOG.md` и `CHANGELOG.md`.
- Концепция интерактивного Digital-аудита для главной кнопки CTA.

- Исправлена работа кнопки «Наверх» в условиях кастомного скролла.
- Настроен универсальный перехват якорных ссылок для плавного скролла.

## [0.0.1] — 2026-02-13
