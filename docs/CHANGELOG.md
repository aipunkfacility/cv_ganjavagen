## [0.7.1] — 2026-02-13

### Изменено

- **UX:** Удалена кнопка «Наверх» для минимизации интерфейса.
- **Theme:** Светлая тема теперь активируется по умолчанию (системные настройки игнорируются при первом посещении).
- **Assets:** Исправлены пути к изображениям и иконке для стабильной работы на GitHub Pages и локально.

---

## [0.7.0] — 2026-02-13

### Добавлено

- **Deployment:** Настроен автоматический деплой на GitHub Pages через GitHub Actions.
- **Config:** Настроены `site` и `base` в `astro.config.mjs` для корректной работы на стандартном домене GitHub.

---

## [0.6.0] — 2026-02-13

### Добавлено

- **Concept:** Блок «База и философия» (Образование и Философия) восстановлен из поврежденного состояния.
- **Capture Compatibility:** Внедрен режим `.no-snap` и `@media print` стили для улучшения работы инструментов захвата скриншотов.

### Изменено

- **UI:** Унифицирована структура блока `Concept` с остальными секциями (`LocalSolutions`, `Method`) для визуальной целостности.
- **Contacts:** Обновлен номер Zalo во всех компонентах.

---

## [0.5.0] — 2026-02-13

### Добавлено

- **Темная тема:** Полноценная поддержка темного режима с ручным переключением.
- **UX:** Сохранение выбора темы в `localStorage` и синхронизация с системными настройками ОС.
- **Performance:** Оптимизированный инлайновый скрипт для предотвращения "вспышки" белого фона.
- **UI:** Плавные CSS-переходы (0.3s) для всех элементов интерфейса при смене темы.

---

## [0.4.0] — 2026-02-13

### Добавлено

- Новый навык агента: `generate-professional-readme` для автоматической генерации профессиональной документации.
- Инструменты статического анализа: `@astrojs/check` и `typescript` для повышения надежности кода.

### Исправлено

- **Critical:** Исправлена ошибка `ReferenceError: t is not defined` в компоненте `LanguagePicker` (отсутствовал импорт хука переводов).
- **Linting & Accessibility:**
  - Указаны явные размеры изображения Hero (1500x2000) для предотвращения сдвига макета (CLS).
  - Добавлены `aria-label` для логотипа и переключателя языков.
  - Оптимизирован HTML-код ключевых компонентов (Hero, LanguagePicker) для исключения ошибок парсинга линтером.

---

## [0.3.0] — 2026-02-13

### Добавлено

- Комплексная мобильная оптимизация (Responsive Design).
- Внедрение `dvh` для стабильной высоты секций на мобильных устройствах.
