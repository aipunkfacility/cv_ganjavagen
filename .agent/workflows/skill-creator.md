# Создание навыков (Skill Creator)

Этот воркфлоу описывает процесс разработки, упаковки и тестирования новых навыков для расширения возможностей агента.

## 1. Подготовка и проектирование

- [ ] Сформулируйте "одну задачу", которую будет решать скилл (Unix Philosophy).
- [ ] Придумайте название в формате `verb-noun` (например, `optimize-images`).
- [ ] Определите входные данные (файлы, переменные, API).

## 2. Создание структуры

- [ ] Создайте директорию для скилла в `.agent/skills/[name]`.
- [ ] Создайте основной файл инструкций `SKILL.md`.
- [ ] Если требуются скрипты, создайте папку `scripts/` и реализуйте их на Python или Node.js.

### 3. Наполнение SKILL.md

- [ ] **Заголовок и описание**: Четко опишите, когда агент должен вызывать этот скилл.
- [ ] **Принципы**: Укажите ключевые правила (лаконичность, отсутствие мета-контекста).
- [ ] **Примеры (Few-Shot)**: Добавьте примеры "Вход -> Выход" для обучения агента.
- [ ] **Workflow**: Опишите внутренние этапы работы скилла.

### 4. Валидация

- [ ] Проверьте размер `SKILL.md` (минимизируйте токены).
- [ ] Убедитесь, что скрипты имеют обработку ошибок.
- [ ] Протестируйте скилл, вызвав его через `mcp_skillport_load_skill`.

### 5. Регистрация

- [ ] Убедитесь, что скилл виден в поиске `mcp_skillport_search_skills`.
